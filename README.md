# Sound Sculpture

音で彫刻するWeb作品のプロトタイプです。

PCに入力された音をWeb Audio APIで解析し、Three.js上の3Dオブジェクトをリアルタイムに変形させます。音による変化は一時的なビジュアライザーではなく、形状へ不可逆に蓄積されます。演奏が止まる、または手動で完了すると、その時点の形状を完成作品として固定します。

## Concept

この作品では、演奏を彫刻行為、音を物質への圧力、時間を形成プロセスとして扱います。

一般的な音楽ビジュアライザーのように音が止まると元に戻るのではなく、音の影響が物質へ定着していくことを軸にしています。

## Features

- マイクまたはPCの音声入力を使用したリアルタイム音解析
- FFT解析による低音・中音・高音の分離
- 低音によるコア形状の累積変形
- GLSL Shaderによる半透明のデジタル外膜
- 中高音に反応するノイズ、フレネル発光、データライン表現
- 高音で生成され、表面へ吸着して固定されるパーティクル
- 一定時間の無音による自動完成
- `完了`ボタンによる手動完成
- 完成後の展示モード
- GLTF出力

## Getting Started

依存関係をインストールします。

```sh
npm install
```

開発サーバーを起動します。

```sh
npm run dev
```

ブラウザで表示されたローカルURLを開き、`音入力を開始`を押してください。ブラウザからマイク入力の許可を求められます。

## Usage

1. `音入力を開始`を押す
2. 音を入力する
3. 低音でコアが変形し、中高音で外膜とパーティクルが反応する
4. 無音が一定時間続くと自動で完成する
5. 任意のタイミングで`完了`を押して手動完成できる
6. 完成後に`GLTF 出力`で形状を書き出す

## Build

本番ビルドを作成します。

```sh
npm run build
```

ビルド結果を確認します。

```sh
npm run preview
```

## Current Prototype Notes

現在はプロトタイプ段階です。音入力はリアルタイム解析のため、同じ曲でも入力環境や音量によって結果が変わる可能性があります。

同じ音源から同じ形を再生成するには、今後以下の調整が必要です。

- 音源ファイル入力への対応
- 固定timestepによる解析
- FFT設定とサンプリング条件の固定
- 変形量の正規化
- パーティクル生成タイミングの決定性向上
